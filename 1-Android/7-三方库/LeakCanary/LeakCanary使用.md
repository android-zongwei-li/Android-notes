# 开始使用

添加依赖

```kotlin
dependencies { 
    // debugImplementation because LeakCanary should only run in debug builds. 
    debugImplementation 'com.squareup.leakcanary:leakcanary-android:2.14' 
}
```

确认LeakCanary正常运行

```kotlin
Logcat查看

D LeakCanary: LeakCanary is running and ready to detect leaks
```
